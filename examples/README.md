# Examples

Downloadable CSV templates (synthetic training data — no client/patient identifiers).

| File | Description |
|------|-------------|
| [calibration_prep_example.csv](calibration_prep_example.csv) | Six-level calibration prep (10 mL, 1000 ng/mL stock) |
| [batch_sequence_example.csv](batch_sequence_example.csv) | Short 1633A-style planning sequence |
| [qc_export_example.csv](qc_export_example.csv) | Batch QC checklist export row |

Generate fresh files from the app tabs (Calibration prep, Batch sequence, Batch QC) for your laboratory after SOP review.
